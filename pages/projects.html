ETS: ESP32 Tracking system

The purpose of this project is to sniff probe request packets sent by smartphones that are looking for a Wi-Fi connection. From each sniffed packet some informations will be extracted and elaborated in order to do several analysis.

The firmware of the ESP is written with the purpose to sniff probe request packets sent by smartphones that are looking for a Wi-Fi connection. From each sniffed packet some informations will be extracted. After each minute these informations are sent to a server that process and store thme in a MySQL database. Then is possible to see the processed information, like position, time frequency and etc., through a GUI.

The system is also able to recognize random generated MAC address: in order to improve security some devices uses random generated MAC address to send probe request packets. Briefly a random generated MAC address can be recognized because the 2nd bit of the 1st byte is set to one. A class is able to exploit a leak in this mechanism and group all tuples with different MAC addresses belonging to the same device.
